HTML Structure

FUNDITO template is based on Bootstrap - framework for developing responsive, mobile first projects on the web. More information you can find at http://getbootstrap.com/

Grid system

Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. It includes predefined classes for easy layout options, as well as powerful mixins for generating more semantic layouts.

Example of usage

If you want to create new container using Bootstrap grid, the basic grid is:

<div class="container">
        <div class="row">
        ...
        </div>
</div>

To create container with white backgroud and top and bottom bordered you can use:

<div class="container-white container-bordered">
    <div class="container">
        <div class="row">
        ...
        </div>
    </div>
</div>

Grid columns are created by specifying the number of twelve available columns you wish to span. For example, three equal columns would use three .col-xs-4.

If you want to create new container using grid, after "row" use div with specified grid class:

<div class="row">
    <div class="col-xs-12 col-sm-6 col-md-8 col-lg-6">-- content --</div>
    <div class="col-xs-12 col-sm-6 col-md-8 col-lg-6">-- content --</div>
    <div class="col-xs-12 col-sm-6 col-md-8 col-lg-6">-- content --</div>
</div>

Useage of class prefix:

  • ".col-xs-" for Extra small devices, Phones (<768px)
  • ".col-sm-" for Small devices, Tablets (≥768px)
  • ".col-md-" for Medium devices, Desktops (≥992px)
  • ".col-lg-" for Large devices Desktops (≥1200px))

CSS Structure

There are 4 CSS files used in FUNDITO:

  • css/bootstrap.css - contains grid system, main css settings, responsive grid and media queries
  • css/font-awesome.css - css file for usage vector icons
  • css/lightbox.css - css for a script used to overlay images on the page
  • css/style.css - main css file
  • css/style.custom.css - your custom css file, change it!
  • css/colors - folder with couple color schemes

The main style.css has a table of contents:

 1. All components
 2. Navigation bar and fix navigation bar
 3. Search bar
 4. Discover box
 5. Container about
 6. Container about
 7. Projects block
 8. Categories block
 9. Info container
 10. Contacts block
 11. Partners block
 12. Portfolio filter
 13. Project page
 14. Footer
 14. Custom media quaries

The CSS files are separated into sections using commented header:

/*all components */

so you can just search by element:)

Projects filter by categorie

In index2.html you can find Projects filter by categorie

If you want to create new categorie - you must just add id of categorie in input and add corresponding value

<div id="portfolio-filter">
    <input id="All" class="btn current btn-info" type="button" value="All">
    <input id="Creative" class="btn btn-info" type="button" value="Creative">
    <input id="Developments" class="btn btn-info" type="button" value="Developments">
    ...
</div>

To connect filter button with project add to the div's portfolio-listing class name of your filter (for example: All Creative). You can add there categories how many you want.

<div class="container projects-box projects">
    <div class="col-lg-3 col-sm-6 col-xs-12 portfolio-listing All Creative block" style="display: block;">
        <div class="project-box">
        ....
        </div>
    </div>
</div>

Color Options

Current color scheme of FUNDITO is red, but you can easy change it - in folder css/colors you will find a few files with blue, green and purple color schemes

Folder contains css file custom.css where you can change colors and create your own page!

To use the color scheme on website, edit html file at line 16 - just delete comments <!-- --> and rename css file.

For example if you need an blue color, replace css/colors/green.css with css/colors/blue.css
